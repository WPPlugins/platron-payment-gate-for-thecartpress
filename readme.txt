Оплата через Платрон в модуле TheCartPress

== Description ==

С помощью Платрон вы сможете принимать платежи более чем через 20 платежных систем, объединенных единым техническим шлюзом и коммерческими условиями подключения.

== Installation ==

1. Загрузите плагин
2. Устанавливаем модуль (Плагины->Установить и Активировать)
3. Регистрируемся на platron.ru
4. Заходим TCP настройки->Платежные методы->platron и заполняем требуемые данные *
5. Для того, чтобы настроить выбор платежной системы на стороне магазина необходимо сделать новый экземпляр модуля оплаты и заполнить поле Payment system
6. Чтобы правильно выводились логотипы ПС (см поле Payment System logo) нужно сделать неактивным первый экземпляр модуля оплаты и либо удалить из него картинку, или поставить общую для всех методов.

* Если не хотите принимать заказ к оплате - поменяйте статус заказа на отменен.
